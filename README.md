# LaundryApp
Website laundry modern dengan layanan antar-jemput, cuci, setrika, dan express.

## Fitur
- Layanan: Cuci + Setrika, Setrika, Cuci Express
- Harga transparan
- Cara kerja mudah
- Formulir kontak langsung ke WhatsApp

## Teknologi
- Next.js
- Tailwind CSS
